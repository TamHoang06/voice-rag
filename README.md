# AI Podcast Agent

**Production-ready AI platform**: Document → Podcast generation with RAG, Gemini 2.5 Flash (LLM/TTS/STT), F5-TTS voice cloning, full-stack Vietnamese podcast workflow.

## 🚀 Features
| Feature | Description |
|---------|-------------|
| **Document Processing** | PDF/DOCX/TXT + image OCR → RAG FAISS vectorstore |
| **Podcast AI** | Gemini auto-segment script + timing estimation |
| **Voice Cloning** | F5-TTS zero-shot (10s sample) + Gemini 9 voices |
| **Interactive Q&A** | Real-time RAG during playback |
| **Voice Library** | Upload → active voice auto-clone |
| **Production** | Rate limiting, Docker, pytest E2E |
